Where Am I

Where Am I was a 2-day collaborative project I (with a few others) created for a hackathon. The project calls an API to fetch data about native tribes and groups that lived in the user's area, so they can find out more about the land's heritage. I created it as a beginner to web development, so I had to learn several intermediate javascript concepts in a matter of days.

Technologies I Used

  • HTML & CSS
  • Javascript
  • Native Land API

Astrodome

Astrodome was yet another hackathon project. The goal was to have the project be the prototype for a global connection system for youth, so that they could interact with like minded peers throughout the world, in a system that would automatically match them up.

Technologies I Used

  • HTML & CSS
  • Javascript
  • Jquery
  • Python
  • Flask & Socketio
  • Bootstrap
